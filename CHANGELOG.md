# Changelog

---

## [1.0.0] - 2026-05-05

Initial release.

- Circular and arc action bar layouts with configurable radius, arc degrees and start angle
- Multiple bars, each independently positioned and configured
- Click-through mode with modifier key override (Ctrl/Alt/Shift)
- Fade-on-hover with configurable alpha and delay
- Action bar paging (modifier keys, stance/form map, custom macro conditionals)
- Visibility conditionals (combat, vehicle, pet, custom)
- Edit Mode integration: bars auto-unlock for repositioning when WoW Edit Mode is open
- Optional Masque skinning support
- Profiles with AceDB
